# Effet de la hausse des taux sur la performance des entreprises

Ce dépôt contient les scripts Python utilisés pour construire l’échantillon, créer l’indicateur de vulnérabilité initiale et estimer les régressions du rapport.

## Contenu
- 01_construction_indicateur_vulnerabilite.ipynb
- 02_regressions_principales.ipynb
- 03_controles_qualite.ipynb

## Objet
Le projet étudie l’effet de la hausse des taux directeurs sur la performance et la structure financière d’un panel international de grandes entreprises cotées sur la période 2022-2025.
